    Guia de Desenvolvimento

    Padrões Arquiteturais

    • Application Factory Pattern: A aplicação Flask é criada via create_app().
    • Repository Pattern: O acesso aos dados é abstraído por classes de repositório (ex: PlcRepo, RegisterRepo).
    • Adapter Pattern: A comunicação com protocolos de CLP é isolada em classes Adapter (ex: ModbusAdapter).
    • Service & Manager Layer: A lógica de negócios é dividida em Services (orquestração de alto nível) e Managers (gerenciamento de estado e workers).

    Iniciar a aplicação

    criar #app #iniciar #flask

    Para dar inicio ao projeto, deve-se criar a aplicação Flask geral também para app_context, seguido o seguinte exemplo

    from src.app import creat_app #importa a criação da apicação

app = creat_app() # cria a aplicação

app.run(host="0.0.0.0", port=5000, debug=True, use_reloader=False) #inicia o servidor Flask

    para configurar vá para [[Configurando aplicação Flask]]

    Adicionar PLCs

    criar #plc #db

    Para adicionar PLCs via código, primeiro deve-se criar o objeto do PLC e com esse objeto, adicionar ao banco de dados sempre usando o app_context

    
from src.app import creat_app 
from src.models import PLC #Import a classe para a criação
from src.repository.PLC_repository import Plcrepo #Import objeto para manipulão no db

app = creat_app()

with app.app_context():
    new_plc = PLC(name='PLCMod', 
    ip_address='127.0.0.1', protocol='modbus', port=5020,
    unit_id=1, is_active=True) #Cria o objeto do PLC
    Plcrepo.add(new_plc) #adiciona plc ao banco de dados



app.run(host="0.0.0.0", port=5000, debug=True, use_reloader=False)

    Adicionar Registradores

    criar #registradores #polling #leitura

    Para adicionar registradores, segue-se a mesma estrutura dos PLCs, importe a classe modelo e adicione com o repository, mas, Register é relacionado ao PLC pelo id do PLC, então precisamos buscar o id do PLC no db e adicionar o id para o register 

    from src.app import creat_app 
from src.models import PLC, Register #importa a class
from src.repository.PLC_repository import Plcrepo 
from src.repository.Registers_repository import RegRepo #importa o objeto para manipulação

app = creat_app()

with app.app_context():
    new_plc = PLC(name='PLCMod', 
    ip_address='127.0.0.1', protocol='modbus', port=5020,
    unit_id=1, is_active=True) 
    Plcrepo.add(new_plc)

    plc = Plcrepo.firts_by(ip_adrres='127.0.0.1') #Pocura o clp no db com um filtro de ip

    reg = Register(plc_id=plc.id,
    name="Registrador_teste,
    address=0,
    register_type="holding",
    data_type="int16",
    scale_factor=1.0,
    offset=0.0,
    unit="°C",
    is_active=True) #Cria o objeto do Register

    RegRepo.add(reg) #adiciona plc ao banco de dados


app.run(host="0.0.0.0", port=5000, debug=True, use_reloader=False)
